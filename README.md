# An-lise-de-vendas-de-chocolate
Este projeto tem como objetivo realizar a análise de dados de vendas de chocolates, utilizando técnicas de ETL, tratamento de dados e visualização analítica para apoiar a tomada de decisão baseada em dados. A base de dados utilizada foi obtida na plataforma Kaggle, contendo informações relacionadas às vendas, produtos, clientes etc...
