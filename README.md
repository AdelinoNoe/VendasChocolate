# Análise de vendas de chocolate
Este projeto tem como objetivo realizar a análise de dados de vendas de chocolates, utilizando técnicas de ETL, tratamento de dados e visualização analítica para apoiar a tomada de decisão baseada em dados. A base de dados utilizada foi obtida na plataforma Kaggle, contendo informações relacionadas às vendas, produtos, clientes, regiões e desempenho comercial.

Na fase de processamento dos dados, foi utilizada a linguagem Python para executar o processo de ETL (Extração, Transformação e Carregamento). Durante esta etapa, foram realizadas atividades como:

-remoção de valores nulos;

-tratamento de dados inconsistentes;

-padronização de formatos;

-eliminação de espaços extras;

-conversão de tipos de dados;

-limpeza e preparação dos dados para análise.

Após o tratamento, os dados foram integrados ao Microsoft Power BI, ferramenta utilizada para criação de dashboards interativos e geração de insights estratégicos. Através das visualizações, foi possível analisar:

-produtos mais vendidos;

-faturamento total;

-desempenho por região;

-tendências de vendas;

-comportamento dos clientes;

-períodos com maior volume de vendas;

-indicadores de desempenho comercial.

O projeto demonstra a aplicação prática de técnicas de análise de dados e Business Intelligence, combinando programação em Python com visualização analítica no Power BI para transformar dados brutos em informações úteis e estratégicas para o negócio.
